## 🔐 시크릿 키 관리 및 실행 가이드

### 1. 리포지토리 클론

### 2. .env 파일 준비
#### .env.example 파일을 최상위 폴더에 .env 이름의 파일로 복사 후 YOUR_OPENAI_KEY 부분에 실제 키 입력

### 3. 실행(IntelliJ 터미널)
#### macOS/Linux : chmod +x start.sh 입력 후 ./start.sh 입력
#### Window : start.bat 입력

### 위 절차대로 .env를 각자 로컬에서 초기 풀 땡기고 한번만 설정하면 이후 start.sh or start.bat 실행만으로 로컬에서 돌릴 수 있음
